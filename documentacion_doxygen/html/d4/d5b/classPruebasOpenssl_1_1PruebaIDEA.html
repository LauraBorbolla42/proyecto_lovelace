bool PruebaIDEA::probarCifradoDescifrado ( )
static

Prueba el funcionamiento de IDEA.

Genea una llave y un vextor de inicialización para cifrar con IDEA en modo CBC. Todo se hace a través de la (supuestamente de alto nivel) interfaz «ENV». Supongo que para tratarse de código de C, sí se puede considerar como «alto nivel»: todo el código de esta función es prácticamente el mismo para cualquier tipo de cifrado por bloques que se quiera usar.

La documentación es muy pobre: la gran mayoría son man pages con títulos raros. Gran parte de este ejemplo salió de las man pages de openssl (la entrada de EVP_aes).

PD: el descifrado no tubo la amabilidad de quitar los bytes que puso de padding el cifrado (cryptopp sí los quita).
